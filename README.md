# GuardAI: Real-Time AI Security System

GuardAI is an advanced security platform designed to detect fake Instagram profiles and cyberstalkers using multi-modal AI analysis.

## Core Features
- **NLP Intent Detection**: Identifies malicious intent in profile bios and messages.
- **Behavioral Analysis**: Flags suspicious patterns (follower/following ratios, spam indicators).
- **Gemini Vision Integration**: Detects AI-generated or suspicious profile images.
- **Explainable AI**: Provides human-readable reasons for risk assessments.
- **Chrome Extension**: Seamlessly scans profiles directly within Instagram.
- **Live Dashboard**: Visualizes risk scores and analysis details.

## Project Structure
- `backend/`: FastAPI server handling AI logic and data processing.
- `dashboard/`: React-based UI for managing and viewing scanned profiles.
- `extension/`: Chrome Extension (Manifest V3) for Instagram integration.
